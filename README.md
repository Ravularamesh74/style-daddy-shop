# 👕 StyleDaddy Shop

🚀 A modern fashion e-commerce web application inspired by the Instagram brand **StyleDaddy**, focused on trendy men's clothing and seamless shopping experience.

---

## 📌 About

StyleDaddy Shop is a stylish and user-friendly online platform that allows users to explore the latest men's fashion collections, including shirts, jeans, sweatshirts, and accessories.

Inspired by real-world fashion branding, this project aims to deliver a smooth UI/UX similar to modern shopping platforms.

---

## 🌐 Live Inspiration

👉 Instagram: https://www.instagram.com/style_daddy1/

---

## ✨ Features

- 🛍️ Browse latest fashion collections
- 🔍 Search & filter products
- 🧾 Product details page
- 🛒 Add to cart functionality
- ❤️ Wishlist feature
- 📱 Fully responsive design
- 🎯 Modern UI inspired by Instagram fashion brands

---

## 🧑‍💻 Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- ⚡ Vite
- 🧠 TypeScript (optional)
